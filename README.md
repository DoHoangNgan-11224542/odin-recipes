# Odin Recipes 🍳

Dự án này là một trang web chia sẻ công thức nấu ăn cơ bản, được xây dựng như một phần của lộ trình khóa học [The Odin Project](https://www.theodinproject.com/). 

Mục tiêu chính của dự án là thực hành và củng cố các kiến thức nền tảng về HTML, đồng thời làm quen với quy trình quản lý mã nguồn bằng Git và GitHub.

## 🚀 Kỹ năng đã áp dụng

Thông qua việc xây dựng dự án này, tôi đã học và thực hành được các kỹ năng:

*   Thiết lập cấu trúc tài liệu HTML5 chuẩn (Boilerplate).
*   Sử dụng các thẻ định dạng văn bản cơ bản như tiêu đề (`<h1>` đến `<h6>`) và đoạn văn (`<p>`).
*   Tổ chức thông tin bằng danh sách có thứ tự (`<ol>`) và danh sách không thứ tự (`<ul>`).
*   Chèn và hiển thị hình ảnh với thẻ `<img>`.
*   Tạo liên kết điều hướng giữa các trang web (Absolute và Relative links) bằng thẻ `<a>`.
*   Sử dụng Command Line/Terminal để quản lý mã nguồn.
*   Thực hiện quy trình làm việc cơ bản với Git (`clone`, `add`, `commit`, `push`).
*   Triển khai (Deploy) một trang web tĩnh lên Internet bằng GitHub Pages.
